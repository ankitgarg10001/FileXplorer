a file explorer for Android users so that they can browse files over android effortlessly and open it in any available application on user’s mobile which is able to process that file.
Some features are following:-
		a) Contains functionalities of many apps under single application which saves memory and space.
		b) Compatibility over a range of android versions.
		c) Interactive user interface/GUI.
		d) Browse most popular file types on your phone at one place.
		e) Interact with every type of file in memory card or phone memory.
		f) Open various files effortlessly.
		
SOFTWARE & HARDWARE REQUIREMENTS
		a) Android 2.3.3 and above
		b) Eclipse Software Development Kit (SDK)
		c) Android Development Kit (ADK)
		d) Android Virtual Device Manager (AVD)
		e) Java Development Kit (JDK)
